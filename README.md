# DevOps E-Commerce Deployment Challenge

## Overview

This project represents a complete deployment of a containerized **E-Commerce Web Application** consisting of:

* Frontend (React)
* Backend (Node.js / Express)
* Database (MongoDB)
* Monitoring stack components

The application is already deployed and functioning in **Microsoft Azure** using a fully containerized architecture.

Your task is to **identify and fix configuration issues** and then successfully **deploy the same solution in AWS (Free Tier)**.

This challenge replicates a **real-world DevOps troubleshooting + cloud migration scenario**.
