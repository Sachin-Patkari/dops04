# StyleVault - Modern Apparel & Accessories

This is a frontend-only React e-commerce application for a modern clothing brand. It features a product catalog, shopping cart functionality, and a simulated checkout process, all managed with Redux Toolkit and styled with Chakra UI.

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone <your-repo-url>
    cd stylevault-app
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    ```
    or
    ```sh
    yarn install
    ```

3.  **Run the application:**
    Start the development server.
    ```sh
    npm start
    ```
    or
    ```sh
    yarn start
    ```
    The application will be available at `http://localhost:3000`.

## Key Features

- **Modern UI**: Built with Chakra UI for a responsive and stylish design.
- **State Management**: Centralized state powered by Redux Toolkit.
- **Client-Side Routing**: Seamless navigation using React Router.
- **Local Data**: All product information is served from a local `products.js` file, simulating an API.
- **Core E-commerce Flow**: Browse products, filter/sort, view details, add to cart, and complete a simulated checkout.
